# Раскладка `unicode` для клавиатуры Lotus 58

Это кастомная раскладка с поддержкой Unicode символов, Home Row модов и других кастомных настроек.

## Прошивка

1.  Запустите `QMK MSYS`.
2.  Перейдите в корневую папку вашей `qmk_firmware`.
3.  Введите команду:
    ```bash
    make tweetydabird/lotus58:unicode:flash
    ```
4.  Когда в консоли появится сообщение `Detecting USB port, reset your controller now...`, переведите клавиатуру в режим прошивки (boot режим).

## Особенности

### Home Row Модификаторы

На базовом слое (0) на домашнем ряду используются клавиши-модификаторы. При быстром нажатии они вводят букву, а при удержании работают как клавиши-модификаторы:

*   **A** = `GUI` (Win/Cmd)
*   **S** = `Alt`
*   **D** = `Ctrl`
*   **F** = `Shift`
*   **J** = `Shift` (правый)
*   **K** = `Ctrl` (правый)
*   **L** = `Alt` (правый)
*   **;** = `GUI` (правый)

### Настройки Энкодеров

*   **Слой 0 (Базовый):**
    *   **Правый энкодер:** Громкость (`+` / `-`)
    *   **Левый энкодер:** Отключен
*   **Слой 1 (Символы):**
    *   **Правый энкодер:** Прокрутка колеса мыши (вверх / вниз)
    *   **Левый энкодер:** Отключен
*   **Слой 2 и 3:** Оба энкодера отключены.

### Комбинации Клавиш (Combos)

*   `m` + `,` = `ъ`
*   `o` + `p` = `х`
*   `e` + `r` = `ё`
*   `'` + `"` = `
*   `(` + `)` = `~`
